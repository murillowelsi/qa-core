# QA Core

A comprehensive knowledge base for software testing fundamentals, practices, and methodologies.

## Structure

- **1-fundamentals/** - Core testing concepts and principles
- **2-test-life-cycle/** - Test levels, types, and lifecycle processes
- **3-test-techniques/** - Various testing techniques and approaches
- **4-test-management/** - Planning, estimation, and risk management
- **5-test-tools/** - Testing tools and their selection criteria
- **6-quality-and-reviews/** - Quality assurance and review processes
- **7-software-processes/** - Software development lifecycle models
- **8-personal-notes/** - Personal insights and reflections

## Usage

Navigate through the folders to explore different aspects of software testing and quality assurance.
